Política de Privacidade

Última atualização: 2026

Esta Política de Privacidade descreve como o ALLINIM coleta, utiliza, armazena e protege os dados pessoais dos usuários, em conformidade com a legislação aplicável, incluindo a Lei nº 13.709/2018 (Lei Geral de Proteção de Dados - LGPD).

1. Dados coletados

Podemos coletar os seguintes dados pessoais:

  • Nome
  • Número de telefone
  • Localização do dispositivo
  • Informações de uso do aplicativo
  • Dados técnicos do dispositivo

2. Finalidade do uso dos dados

Os dados coletados são utilizados para:

  • Criar e gerenciar contas
  • Conectar usuários a prestadores de serviço
  • Calcular distâncias e estimativas
  • Melhorar a experiência do usuário
  • Garantir a segurança da plataforma

3. Base legal

O tratamento de dados é realizado com base no consentimento do usuário e na execução de contrato, conforme previsto na LGPD.

4. Dados de localização

O ALLINIM pode coletar dados de localização em tempo real para possibilitar a oferta de serviços próximos ao usuário.

5. Compartilhamento de dados

Os dados poderão ser compartilhados com prestadores de serviços quando necessário para a execução da solicitação realizada pelo usuário.

6. Armazenamento e segurança

Adotamos medidas técnicas e administrativas adequadas para proteger os dados pessoais contra acessos não autorizados, perda ou uso indevido.

7. Retenção de dados

Os dados serão armazenados pelo tempo necessário para cumprimento das finalidades descritas nesta política e obrigações legais.

8. Direitos do usuário

O usuário poderá, a qualquer momento:

  • Solicitar acesso aos dados
  • Corrigir dados incompletos ou desatualizados
  • Solicitar exclusão de dados pessoais

9. Exclusão de conta

O usuário pode solicitar a exclusão de sua conta diretamente pelo aplicativo ou entrando em contato pelo e-mail informado abaixo. A exclusão será realizada em prazo razoável, respeitando obrigações legais.

10. Alterações nesta política

Esta Política de Privacidade poderá ser atualizada periodicamente.
